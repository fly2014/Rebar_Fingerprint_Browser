Rebar_Browser

钢筋浏览器（Rebar Fingerprint Browser）

A desktop tool for multi-account management and browser environment isolation, designed for scenarios requiring multiple instance logins, independent environments, and automation workflows.

🌐 Project Overview

Rebar Browser provides multi-instance browser management capabilities on Windows, using the built-in Chromium engine (Chrome / Edge).

It is mainly used for:

Multi-account login management
Browser environment isolation
Automation support
Enterprise / team multi-instance workflows
🧩 Core Features
👤 Multi-account management
Each tab runs an independent browser environment
Multiple accounts can be logged in simultaneously
Complete data isolation between accounts
🧱 Environment isolation
Independent cookie storage
Independent LocalStorage
Browser fingerprint isolation (Canvas / WebGL, etc.)
Each instance runs independently
🌍 Network support
Supports HTTP / HTTPS / SOCKS5 proxies
Different proxy per instance
Global network switching
🧰 Automation capabilities
Command execution support
Action recording & replay
Multi-window synchronized operations
🔌 Extension support
Chrome extension support
Load unpacked extensions
Fully compatible with Chromium ecosystem
💻 Supported systems
Windows 7 / 10 / 11
Runs on system Chrome / Edge engine
