# Rebar_Browser
[钢筋指纹浏览器（Rebar Fingerprint Browser）](https://gjllq.com/)

一款用于多账号管理与浏览器环境隔离的桌面工具，适用于需要多实例登录、独立环境运行和自动化操作的场景。

🌐 项目简介

钢筋指纹浏览器 提供多浏览器实例管理能力，支持在 Windows 平台运行，并可调用系统内置 Chromium 内核（Chrome / Edge）。

主要用于：

多账号登录管理
浏览器环境隔离
自动化操作支持
企业 / 团队多实例协作场景
🧩 核心功能
👤 多账号管理
每个标签页独立运行浏览器环境
支持同时登录多个账号
账号之间数据完全隔离
🧱 环境隔离
Cookie 独立存储
LocalStorage 独立
浏览器指纹隔离（Canvas / WebGL 等）
每个实例互不影响
🌍 网络支持
支持 HTTP / HTTPS / SOCKS5 代理
可为不同实例配置不同网络环境
支持全球网络切换
🧰 自动化能力
支持命令执行
支持操作回放（脚本记录与执行）
支持多窗口同步操作
🔌 扩展支持
支持 Chrome 扩展程序
支持解压安装扩展
兼容 Chromium 内核生态
💻 支持系统
Windows 7 / Windows 10 / Windows 11
基于系统 Chrome / Edge 内核运行
