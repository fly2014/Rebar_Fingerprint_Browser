Rebar_Browser

[钢筋浏览器（Rebar Fingerprint Browser）](https://fr.gjllq.com/)

Un outil de bureau pour la gestion multi-comptes et l’isolation des environnements navigateur, conçu pour les scénarios nécessitant plusieurs instances et l’automatisation.

🌐 Présentation

Rebar Browser permet de gérer plusieurs instances de navigateur sous Windows, basé sur le moteur Chromium (Chrome / Edge).

Utilisé principalement pour :

Gestion de plusieurs comptes
Isolation des environnements navigateur
Support de l’automatisation
Utilisation en équipe ou entreprise
🧩 Fonctionnalités principales
👤 Gestion multi-comptes
Chaque onglet possède un environnement indépendant
Connexion simultanée à plusieurs comptes
Isolation complète des données
🧱 Isolation de l’environnement
Cookies indépendants
LocalStorage séparé
Isolation de l’empreinte navigateur (Canvas / WebGL)
Instances totalement indépendantes
🌍 Réseau
Support HTTP / HTTPS / SOCKS5
Proxy différent par instance
Changement de réseau global
🧰 Automatisation
Exécution de commandes
Enregistrement et lecture d’actions
Synchronisation multi-fenêtres
🔌 Extensions
Support des extensions Chrome
Installation d’extensions décompressées
Compatible Chromium
💻 Systèmes supportés
Windows 7 / 10 / 11
Basé sur Chrome / Edge système
