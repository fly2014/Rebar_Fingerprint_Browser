Rebar_Browser

[钢筋浏览器（Rebar Fingerprint Browser）](https://jp.gjllq.com/)

マルチアカウント管理とブラウザ環境分離のためのデスクトップツールで、複数インスタンスログインや自動化操作に対応しています。

🌐 概要

Rebar Browser は Windows 上で複数ブラウザインスタンスを管理できるツールで、Chromium（Chrome / Edge）エンジンを利用しています。

主な用途：

複数アカウント管理
ブラウザ環境の分離
自動化処理のサポート
企業・チームでの多インスタンス運用
🧩 主な機能
👤 マルチアカウント管理
各タブが独立したブラウザ環境
複数アカウント同時ログイン可能
データ完全分離
🧱 環境分離
Cookie 分離
LocalStorage 分離
ブラウザ指紋（Canvas / WebGLなど）分離
各インスタンスは独立動作
🌍 ネットワーク
HTTP / HTTPS / SOCKS5 プロキシ対応
インスタンスごとにプロキシ設定可能
グローバル切替対応
🧰 自動化機能
コマンド実行
操作録画・再生
マルチウィンドウ同期操作
🔌 拡張機能
Chrome拡張対応
解凍拡張の読み込み
Chromium互換
💻 対応OS
Windows 7 / 10 / 11
システムの Chrome / Edge を利用
