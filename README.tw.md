Rebar_Browser

[鋼筋指紋瀏覽器（Rebar Fingerprint Browser）](https://tw.gjllq.com/)

鋼筋指紋瀏覽器 一款用於多帳號管理與瀏覽器環境隔離的桌面工具。

🌐 專案簡介

Rebar Fingerprint Browser 支援在 Windows 上運行多瀏覽器實例，基於 Chromium（Chrome / Edge）核心。

主要用途：

多帳號管理
瀏覽器環境隔離
自動化操作
團隊/企業多實例使用
